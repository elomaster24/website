Browser

Was ist ein Browser? Ein Browser ist eine Software für PC oder Mobilgeräte, welches einem erlaubt, das World Wide Web (WWW) aufzurufen. Der Browser ist auch dafür da, um den HTML-Code einer Webseite darzustellen.

Geschichte des Browsers.

Der erste Browser auf dem Markt hieß Mosaic welcher von Marc Andreesen entwickelt wurde. Mosaic ist auch der Vorläufer von dem Mozilla Firefox wie wir es heute kennen. 1994 gründete Marc Andreesen Netscape und veröffentlichte Netscape-Navigator, welcher der erste Browser war, der für alle Menschen verfügbar war. Ein Jahr später brachte Microsoft ihren Internet Explorer heraus, welcher auch der Anfang des Browserkrieges war. Ein paar Jahre später gründete die Firma Netscape Firefox, welche dann Mozilla Firefox in 2002 entwickelten. In 2008 wurde dann der heutige Marktführer heraus gebraucht, nämlich Google Chrome welcher von 67 % aller Computer-Usern benutzt wird.

Wie funktioniert ein Browser?

Die Funktion des Browsers ist es hauptsächlich HTML-Dokumente zu visualisieren. Dies macht er in dem er von dem PC, auf dem der Browser läuft, eine IP für eine Webseite bekommt. Mit dieser IP sendet der Browser eine Anfrage an die Webseite, diese Webseite bearbeitet dann die Anfrage und gibt die Ressourcen der Webseite also HTML-Code, Bilder usw. zurück an den Browser. Danach gehen diese Ressourcen in den Rendering Engine, der Rendering Engine analysiert und wandelt die Daten dann in ein Format um, welches weiterverarbeitet werden kann. Mit den umgewandelten Daten kann dann der UI-Backends weiter arbeiten, dieser visualisiert einfach die Webseite.

Welche Browser gibt es?

Heutzutage gibt es sehr viele Browser, dabei gibt es aber nur einige, die wirklich richtig benutzt werden. Der meistbenutzte Browser wäre Google Chrome mit 67 % Platz 2. wäre dann Apples Safari mit 9,6 % knapp dahinter ist Microsoft Edge (vorher Internet Explorer) mit 9,3 % dann kommen noch Mozilla Firefox und Opera mit jeweils 3,6 % und 2,3 %.

Die Browserkriege

Die Browserkriege wurden damals zwischen Netscape und Internet Explorer geführt (Microsoft). Es hat alles begonnen damit, dass Microsoft den alten Mosaic-Code lizenziert hat. Darauf haben sie dann Internet Explorer entwickelt. Dies sorgte dafür, dass beide Firmen sehr stark sich gegenseitig übertrumpfen wollten. Netscape brachte dadurch das heutige JavaScript raus und Microsoft reagierte damit das sie CSS rausbrachten. 1997 baute das Microsoft Team ein riesiges E, welches sie zur Veröffentlichung von Internet Explorer 4.0 vor den Headquarters von Netscape aufstellten. Später startete Microsoft, Internet Explorer mit Windows zu liefern, was dazu führte, dass sie innerhalb von 4 Jahren 75 % des Marktes hielten und 1999 sogar 99 %. Daraufhin gründete Netscape Mozilla welche dann 2002 Mozilla Firefox rausbrachten bis 2010 hatten Sie und andere Internet Explorers Marktanteil auf 50 % reduziert.
